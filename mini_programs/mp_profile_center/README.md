# Mp Profile Center

Tracked Mp JSON migration fixture for the first-party super host.

It covers:

- basic Mp layout, image, card, and button rendering
- internal mini-program navigation
- bundled asset delivery through `hosts/super_app_host`

Build it with:

```powershell
dart run ..\..\packages\mini_program_tooling\bin\miniprogram.dart build --mini-program-root .
```
